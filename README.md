# API-Project

## StockTrack

A simple stock market tracker built for the Web Dev Full Stack Project.

## Features

Search any stock ticker (AAPL, TSLA, MSFT, etc.)
View current price, % change, and a historical price chart
Save tickers to a personal watchlist
Watchlist persists across reloads (stored in localStorage)

## Tech

HTML / CSS / vanilla JavaScript
Alpha Vantage API for stock data
Chart.js for price charts
localStorage for watchlist persistence
